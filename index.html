About This Research

Study Area
Ubungo Municipality, Dar es Salaam — 14 wards, urban core to peri-urban fringe
Study Period
2022–2025 · 488 geo-referenced observations
Model
GTWR · R² = 0.8912 · Adj R² = 0.8900 · AIC = −1,166.7
Key Predictors
Night-time light (r = +0.884) · Population density · BRT, road & CBD accessibility
Key Finding
Median +71% (775k→1.325M TZS). Core–periphery ratio: 14.7× (Sinza vs Kibamba)
Projection
2040 scenario — WNW & NE corridors highest value tier (≥25M TZS)
Data Source
Ministry of Lands · OSM road network · NASA VIIRS nighttime light · DART terminals

About This Dashboard
This research dashboard was developed as a supplementary output to the Bachelor's dissertation submitted to the Department of Geospatial Science and Technology, Ardhi University, Dar es Salaam, 2026. All spatial data processing and cartographic output was carried out in ArcMap 10.8. Statistical modelling, covariate construction, and 2040 projections were implemented in Python 3 using the pandas, numpy, scipy, statsmodels, and mgwr libraries. The dashboard is hosted on GitHub Pages and is accessible at DEBRUYNE007445.github.io/GTWR-RESEARCH-DASHBOARD.
